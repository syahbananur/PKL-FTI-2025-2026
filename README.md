# Sistem Informasi Inventaris Barang

Aplikasi berbasis Web untuk mengelola stok barang masuk dan keluar, mencatat supplier, dan tujuan distribusi.

## Fitur Utama
- **Multi-User:** Login sebagai Admin dan Staff Gudang.
- **Manajemen Stok:** Stok bertambah saat barang masuk, berkurang saat keluar.
- **Laporan Otomatis:** Rekapitulasi bulanan.
- **Master Data:** Supplier, Satuan, Kategori, Tujuan.

## Cara Instalasi
1. Clone repository ini.
2. Import file `database/db_inventaris.sql` ke phpMyAdmin.
3. Sesuaikan koneksi di `config/koneksi.php`.
4. Buka di browser.
